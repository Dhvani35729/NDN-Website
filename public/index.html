How does this breakthrough networking work?

WandServers are build upon Named Data Networking (NDN). The Named Data Networking (NDN) project aims to develop a new Internet architecture that can capitalize on strengths — and address weaknesses — of the Internet’s current host-based, point-to-point communication architecture.
